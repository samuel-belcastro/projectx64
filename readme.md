# PROJEX64

PROJect Emulator X64

## Overview

This project is a personal exploration of low-level computing concepts implemented in Rust. My goal is to deepen my understanding of how software interacts with hardware by building a simple CPU emulator from first principles—covering memory models, instruction encoding, control flow, and execution.

I’m using Rust intentionally for its strong safety guarantees and emphasis on correctness, which aligns well with a security-first mindset. This work complements my day-to-day engineering by strengthening fundamentals that are broadly applicable to systems programming, secure software design, and performance-critical environments.

Longer term, I’m interested in expanding my knowledge of how these concepts apply in real-world domains such as embedded systems, IoT devices, and vehicle software—areas where reliability, safety, and efficiency are especially important.

This project is part of my ongoing professional development and curiosity as an engineer.
